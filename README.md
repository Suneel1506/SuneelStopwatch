# SimpleStopwatch

A simple stopwatch application built with **Swift**. This app allows users to measure time either in seconds or milliseconds. 
It includes features such as start, pause, and restart buttons, providing a user-friendly way to keep track of time.

## Features
- **Selectable Time Units:** Choose between seconds or milliseconds to track time.
- **Start Button:** Begin timing with the click of a button.
- **Pause Button:** Pause the stopwatch at any time.
- **Restart Button:** Reset the stopwatch back to zero and start again.

## Technologies Used
- **Swift** (Programmatically built without Storyboard or Interface Builder)
- **UIKit** for the user interface
